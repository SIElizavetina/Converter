Преобразовать число из любой системы счисления в десятичную можно следующим образом: 
каждый разряд числа необходимо умножить на X в спени n, 
где X - основание исходного числа, n - номер разряда. 
Затем суммировать полученные значения.
abc = (a*x^2 + b*x^1 + c*x^0 )
Примеры:
567(8) = (5*8^2 + 6*8^1 + 7*8^0 ) = 375(10)
110(2) = (1*2^2 + 1*2^1 + 0*2^0 ) = 6(10)
A5(16) = (10*16^1 + 5*16^0 ) = 165(10)
